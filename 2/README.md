This is one of the simplest star patterns. We need to form a right-angled triangle where the number of stars in each row increases line by line. Row i contains exactly i + 1 stars.

1.Run an outer loop from 0 to N-1 to handle rows.                                                                                                                                                                                           
2.For each row i, run an inner loop from 0 to i.                                                                                                                                                           
3.In the inner loop, print a star (\*).                                                                                                                                                          
4.After finishing the stars of one row, move to the next line using endl.
